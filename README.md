## 📌 Sobre o Repositório

Este repositório funciona como um portfólio e laboratório pessoal para o desenvolvimento de soluções do lado do servidor. Aqui explorarei desde sistemas de autenticação seguros até lógicas de negócio complexas e microsserviços escaláveis, sempre prezando pelas melhores práticas de segurança e estruturação de código.

## 🚀 Como Executar os Projetos

Devido à natureza diversa deste repositório, cada projeto possui suas próprias dependências e comandos. No entanto, o fluxo geral é:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/PedroPEPEUHenrique/APIs-BackEnd.git](https://github.com/PedroPEPEUHenrique/APIs-BackEnd.git)
    cd APIs-BackEnd
    ```

2.  **Acesse o projeto desejado:**
    ```bash
    cd Nome-do-Projeto
    ```

3.  **Siga o arquivo `README.md` específico dentro da pasta do projeto** para instruções detalhadas de instalação de dependências e execução do servidor.
